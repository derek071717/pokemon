# BOLT
## 簡介
Bolt是一款免費將程式碼「可視化」的套件，適合用於「教學」及「初學者」。

## 2D遊戲試做
### 套件下載
- Bolt
- Bolt Kit: Platformer Tutorial Assets
- 
### 放置內容物
- Player
- Enemy
- Objective
- PlatformLarge
- Canvas
- GameObject Left 
- GameObject Right
### 程式碼製作
- Player
  - 呼叫Horizontal內建移動套件
  - 設定Player之Variables → Speed為5 型態Float
  - 使用 按鍵移動x速度值 → 移動距離值 
  - Bolt允許動態創建變量並存取值 → 把值轉為座標即可移動
